Odoo Inventory Warehouse Stock Request App

Odoo Warehouse Stock Request App allows users to create new stock requests directly, specifying the required items, quantities, and any additional information related to the request. Users can track the status of their stock requests in real-time. This Odoo app provides visibility into the approval stage and fulfillment progress. The manager can approve the stock request of the user. Users and managers can send the mail of stock requests and generate stock request information pdf reports.

Warehouse Stock Request

User can create stock request and manager can approve or done stock request.

Send Mail

User and manager can send th stock request mail.

Stock Request Report

User and manager can generate stock request pdf report.
