Product Low Performance Sale Report Odoo Apps

Low Sales Report The Odoo app assists users in generating reports for under performing products. Users can customize default criteria, specifying product quantity and amount to generate Sales Low Reports. This Odoo app allows users to select the report type from products, product variants, and product categories within specified date ranges. Generated reports can be conveniently in PDF and XLS formats. Additionally, users can visualize and analyze low-sales reports through a pivot view, providing valuable insights into product performance.

Features

Configure Criteria for Low Sales

User can set default criteria leval of quantity and amount for low sales report.

Set Report Type for Low Sales Report

User can set report type like product, product variant and product category for low sales report.

Set Date Range for Low Sales Report

User can set start date and end date to print low sales report.

Low Sales Report in Pivot View

User can see low sales report in pivot view.

Low Sales Report in PDF Format

Configuration for Low Sales Report

Goto Sales -> Configuration -> Settings under low sales report, User can set default critical level quantity and amount.

Low Sales Report Menu

Goto Sales -> Reporting -> Low Sales Report menu, Once user click on this menu, Low Sales Report wizard will be open.

Low Sales Report Wizard

Once open wizard, User have to select report type like Product, Product Variant and Product Category.

User have to select date range, view quantity and amout as per configured and also they can update quantity and amount.
