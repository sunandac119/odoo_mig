Take a U-Turn and eliminate the printing hassle in POS!!

Every day, numerous documents are printed in any business. Normally you first download the file in your system, issue the print command, choose the format and send it to the printer. Repeating the process with every order is tedious and inefficient for a POS user.

Odoo POS Network Printer Module allows you to directly print the order receipts in POS. It facilitates cost-effectiveness and faster processing of the orders in the Odoo by eliminating the process of downloading them.

Downloading each and every order receipt on a daily basis is hectic and takes up system space too that in turn slows down your system too.

The module Print these receipts without any need for POS Box. The POS Box is slow in nature and even slows down the payment process too. This module shortens the process and allows you to produce a hard copy of any receipt in just a click.

Prerequisites

QZ Tray Client:

  • You need to install the QZ Tray client as the module uses the QZ Tray client for sending direct print commands to the printer. Follow the below link to install the QZ Tray Client- https://qz.io/download/

Noteworthy Points

  • Make sure that the Qz Tray Client is installed on your system before printing so it can connect to the Network Printer.
  • The POS user needs to make sure that the name of the network printer entered in the created printer POS Network record should exactly match the name of the printer connected to the network.
  • The QZ Tray Client uses the exact name to locate the Printer in the network and send the printing commands.

Calling QZ Tray For Printing

  • Start the QZ Tray CLient on your system.
  • Now, open the form in POS you wish to print out. Click on the highlighted Print button on the top center of the form view.
  • It automatically calls the QZ Tray client to print the report on the basis of the configuration settings in the Odoo.

Network Printer Connection Status in POS

The Odoo backend shows the connection status of the set Network Printer to the user. The color of the printer icon indicates the current connection status of the printer.

  • Green: Connection Established

    • At Odoo backend, the QZ Tray Client automatically prompts the user to connect to the Network Printer. If the user allows the website to connect with the printer through the QZ Tray client and the connection is successfully established, the color of the icon turns green.

  • Yellow: Waiting For Printer Connection

    • The yellow printer icon indicates that the Odoo website is connected to the QZ Tray client but is not connected to the Network printer.

  • Red: Disconnected

    • If the printer icon on the Odoo website is red, it means the Odoo website was not able to connect with the specified printer. Make sure that the QZ Tray Client is running on your system.

Detailed Features List

Print POS Networks without downloading pdf in POS

  • Odoo POS Network Printer module allows you to configure ESC/POS network printer with Odoo POS.
  • Once configured, the Odoo POS automatically sends the printing command to the set network printer using the QZ Tray Client while printing an order receipt.
  • The plugin eliminates the need to use the POSBox to connect the printer.

Print reports or any other kind of document directly

  • It allows the POS user to print POS order receipts without using POS Box.
  • The POS user can directly access the printer with the print button on the form view to print it.
  • Users can print the order receipts via the ESC/POS Network Printer.

Print the documents through Network Printer in POS

  • The module currently supports Network printers.
  • The POS user can also set a default Network printer for the QZ Tray to use automatically on each reload. It is configurable.

Manage the configuration from the Odoo backend

  • Being the admin, you can enable/disable the synchronization of the printer with the website from the backend.

Configuring Network Printer

User can specify the ESC/POS Network Printer name in the Point Of Sale configuration form under the IotBox / Hardware Proxy section.Make sure the Printer's name is exactly the same as the name of the printer is used by the QZ Tray client to make connection and send the print commands.
SC/POS Network Printer Connection

POS automatically connects to the configured printer when it is loaded. There is a connection notification icon that the user can check for the connection status. User can also click on the connection notification icon if he wishes to refresh the connection.

When connection status is Red. This means that the POS was unable to connect to the specified printer. Make sure you have the Qz Tray client setup and its running .

When connection status is Yellow . This means that the POS was able to connect to the Qz Tray client on your system and now needs to connect to the configured ESC/POS printer.

When connection status is Green. This means that the POS has successfully established the connection with the configured printer via the Qz Tray Client.

Printing the Receipt

POS automatically sends the print command to the ESC/POS Network Printer when the order is validated.User can print the receipts again from the Receipt Screen if he/she wishes to.
